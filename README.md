# 🧵 Loom — Where Threads Interweave

A next-generation dating platform designed to replace passive swiping with **active, signal-based connection**.

> Stop swiping. Start weaving.

---

## 🚀 Overview

Loom reimagines how people connect online. Instead of waiting in silence after expressing interest, Loom introduces **real-time signals, structured interaction, and progressive connection**.

Unlike traditional apps, Loom is built to:
- Reduce ghosting
- Eliminate awkward conversations
- Encourage meaningful connections
- Remove paywalls from core experiences

---

## ❗ The Problem

Modern dating apps suffer from three major issues:

- **The Waiting Game**  
  You swipe right… and nothing happens.

- **The Dead Opener Problem**  
  Conversations start with “hey” and go nowhere.

- **Paywalled Depth**  
  Real features are locked behind subscriptions.

📊 Key stats:
- 56% users quit within 90 days  
- <1% matches lead to real dates  
- ₹6.2B projected India market by 2028

## ✨ Core Features

### 1. 🔔 Signal Notification (Before Match)
- Anonymous alert when someone likes you  
- No identity revealed until mutual match  
- Eliminates silent uncertainty  

---

### 2. 💬 Simultaneous Q&A Icebreaker
- Both users answer the same questions  
- Answers revealed **only after both respond**  
- Creates instant shared context  

---

### 3. 🗓️ Availability Overlap Engine
- Share simple time slots (morning/afternoon/evening)  
- Detects mutual availability  
- Suggests real connection windows  

---

### 4. 🌱 Progressive Intimacy Levels

| Level | Name   | Unlocks |
|------|--------|--------|
| 🌱 | Thread | Match + signal |
| 🌿 | Weave  | Q&A + chat |
| 🌸 | Knot   | Voice + scheduling |
| 🌺 | Braid  | Video + sharing |
| ✨ | Loom   | Full connection |

➡️ Features unlock through **interaction, not payment**

---

### 5. 📊 Live Thread Score
- Dynamic compatibility score  
- Updates based on:
  - Q&A responses
  - Interaction quality
  - Availability overlap  

---

### 6. 🌿 Slow Loom Mode
- Consent-based pacing  
- No pressure, no forced progression  
- Mutual agreement required
